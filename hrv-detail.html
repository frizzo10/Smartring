How This Is Computed
This is not the ring's own HRV reading — that one comes back undecoded every time. This number is computed independently: raw PPG samples from the ring's raw sensor stream → bandpass filter (0.5–4 Hz) to isolate the pulse → peak detection with sub-sample interpolation → beat-to-beat (RR) intervals → artifact rejection (drops any interval that deviates more than 20% from its local neighborhood) → RMSSD, the root mean square of successive RR-interval differences. It's the same standard time-domain HRV metric clinical devices use, just run on our own raw data instead of trusting a black-box output.
